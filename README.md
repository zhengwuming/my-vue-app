# my-vue-app

## Build Setup


```bash
# clone the project
git clone https://github.com/zhengwuming/my-vue-app.git

# enter the project directory
cd my-vue-app

# install dependency
npm install

# develop
npm run serve
```

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```

# code format check
npm run lint

# code format check and auto fix
npm run lint -- --fix
```
